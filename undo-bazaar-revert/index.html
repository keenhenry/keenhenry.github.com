Undo Bazaar Revert

24 Nov 2013
Tweet

When you accidentally revert all your code changes with bazaar, how to undo the revert?

The solution is quite simple. bazaar leaves a file with .~1~ extension to the reverted file. Therefore to recover the changes itâ€™s simply a copy of this file onto the reverted one.

And, the first time you revert a file it adds .~1~ to the end, if you make more changes and revert again it adds .~2~ and so on.
