Support Wheel Distribution in Python

17 Aug 2014
Tweet

Wheels are new Python binary distribution format, they are intended to replace eggs.

To support distributing wheel, there are a few things you need to do, first, make sure wheel is installed:

$ pip install wheel

Secondly, for a python-only and a 2-and-3 compatible project, you will need a setup.cfg file in the same directory as setup.py, with its contents as following:

[bdist_wheel]
universal = 1

Lastly, build your project with the following command:

$ python setup.py sdist bdist_wheel
