Tab Auto-Completion in Vim

26 Aug 2012
Tweet

Ever wonder how to do words auto-completion in Vim? Here is how:

set wildmode=longest,list,full  " match the longest common string
set wildmenu                               " turn on tab completion in Vim

Having above settings in your .vimrc file, you will magically be able to do words auto-completion with a single Tab keystroke.

To find out more, try :help wildmenu in Vim.
