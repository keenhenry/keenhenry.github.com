Create Tag Clouds Using Jekyll Plugin

30 Dec 2012
Tweet

In this post, I used a Jekyll plugin to create tags for my website; however, those tags were rendered as a list of tags on the web page, and it was not a visually pleasant design.

Therefore, just this morning, I installed another plugin and tweaked it a little bit to collect all the tags in my blog posts into one big tag cloud. The plugin is well-written and well-documented, which makes it easy to use. It is basically creating a ‘tag_cloud’ tag in the Liquid template system.

The resulting tag cloud can be found here. And, through tweaking this Ruby plugin, I actually learned a bit of Ruby, especially about how to sort an Array, and also the differences between sort and sort_by.
